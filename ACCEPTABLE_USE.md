# Acceptable Use

This repository is a public distribution index for Homebrew metadata.

Do not use this tap to publish or promote software that:

- misrepresents an upstream release or artifact digest;
- bypasses provider billing, quota, attribution, or terms of service;
- includes secrets, credentials, private logs, or private provider sessions;
- impersonates official Kungfu Systems release channels.

Formulae and casks should point to public release artifacts with reviewable
evidence.
